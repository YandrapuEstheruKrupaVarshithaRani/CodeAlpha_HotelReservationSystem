🏨 Hotel Reservation System

📌 Project Overview

The Hotel Reservation System is a console-based Java application designed to manage hotel room reservations efficiently. It allows users to view room availability, search rooms by category, make reservations, cancel bookings, and view customer booking details.

This project demonstrates the implementation of Object-Oriented Programming (OOP) concepts and data management using Java.

✨ Features

- View all available and booked hotel rooms
- Search rooms based on categories (Standard, Deluxe, Suite)
- Book a room using customer details
- Prevent duplicate room reservations
- Cancel existing bookings
- Display complete booking records
- Interactive menu-driven user interface

🛠 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList Collection Framework
- Classes and Objects
- Encapsulation
- Constructors and Methods
- Conditional Statements and Loops
- Scanner Class for User Input

📂 Project Structure

- Room Class – Manages room details and availability status
- Booking Class – Stores customer and reservation details
- Hotel Class – Handles room management and booking operations
- HotelReservationSystem Class – Provides the main user interface

🚀 How to Run

1. Install Java JDK on your system.
2. Save the file as "HotelReservationSystem.java".
3. Compile the program:
   javac HotelReservationSystem.java
4. Run the application:
   java HotelReservationSystem
